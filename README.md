# java-learning-path
## Goal
The purpose of these tasks was to master the fundamental building blocks of programming, focusing on **Clean Code** and **Logical Structure**.

## Highlights
* **Logic & Control Flow:** Basic algorithms using loops and conditionals.
* **Data Handling:** Working with arrays and input/output.
* **Problem Solving:** Small console-based tools to solve mathematical or logical riddles.

## Tech Stack
* **Language:** Java 17+
* **IDE:** IntelliJ IDEA
